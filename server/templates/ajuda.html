API de análise de código em Python

Uso:

  • GET /ranking
    ->retorna o ranking de usuário armazenado no banco de dados

  • GET /ranking/repos
    ->retorna o ranking de repositórios armazenado no banco de dados

  • GET /analysis/url_do_repositorio
    ->Realiza uma análise do código presente repositórios e todos seus subdiretórios, juntamente com seus erros

  • DELETE /del/login_usuário
    ->Deleta um usuário
