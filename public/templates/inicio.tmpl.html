Qualidade do código fonte Python

O Pyquali (Qualidade de código fonte em Python) é uma aplicação que realiza a análise de repositórios Python do GitHub, utilizando a ferramenta Pylint e gera um classificação para seus usuários e repositórios.

Pylint realiza várias verificações dentre elas: Padrão de Codificação, Detecção de erro, Ajuda de refatoração, entre outras

O Pylint realiza análise baseado no Guia de Estilo PEP 8 (convenção com padrões de codificação), e atribui uma pontuação a cada repositório analisado, pontuação essa que tem como maior nota a 10.
A visualização dos rankings é liberada para qualquer usuário ou visitante da página, já para a realização da análise dos repositórios são obrigatórios o login e autenticação no GitHub.
